# Lista de APIs Públicas 

Esta lista contiene **APIs públicas que devuelven datos directamente desde la URL**, ideales para prácticas, ejemplos y clases de frontend o backend.

---

## 1. Universidades por país
**URL**
```
http://universities.hipolabs.com/search?country=Argentina
```
**Descripción**  
Listado de universidades con nombre, dominio y sitio web.

---

## 3. Feriados oficiales
**URL**
```
https://date.nager.at/api/v3/PublicHolidays/2026/AR
```
**Descripción**  
Feriados oficiales de un país y año determinado.

---

## 4. Cervecerías
**URL**
```
https://api.openbrewerydb.org/v1/breweries
```
**Descripción**  
Información sobre cervecerías (nombre, ciudad, país, tipo).

---

## 5. Criptomonedas
**URL**
```
https://api.coinlore.net/api/tickers/
```
**Descripción**  
Datos de criptomonedas: precio, market cap, volumen.

---

## 6. Películas de Studio Ghibli
**URL**
```
https://ghibliapi.vercel.app/films
```
**Descripción**  
Películas con título, director, año y descripción.

---

## 7. Usuarios aleatorios
**URL**
```
https://randomuser.me/api/?results=5
```
**Descripción**  
Usuarios ficticios con nombre, email, dirección y foto.

---

## 8. Libros clásicos
**URL**
```
https://gutendex.com/books
```
**Descripción**  
Libros del Proyecto Gutenberg con autor y formatos.

---

## 9. Recetas de comida
**URL**
```
https://www.themealdb.com/api/json/v1/1/search.php?s=
```
**Descripción**  
Recetas con ingredientes, instrucciones e imágenes.
